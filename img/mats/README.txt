Put material images in this folder.

Naming format:
- [material]_[type].jpg
- use lowercase letters and underscores

Examples:
- granite_ingot.jpg
- granite_gem.jpg
- sandstone_ingot.jpg
- spruce_wood.jpg

How names are mapped from the app:
- "Granite Ingot" -> granite_ingot.jpg
- "Gold Gem" -> gold_gem.jpg

If an image file is missing, the app will still work and just hide that icon.
